# bevops-ai-forecast-reorder

## Problem
Beverage companies face demand volatility across SKUs and channels, leading to stockouts or excess inventory.

## Solution
An end-to-end forecasting and reorder recommendation system using historical sales data.

## Deliverables
- SKU-level demand forecasts
- Inventory reorder recommendations
- KPI dashboard for stakeholders

## Tech Stack
Python, pandas, scikit-learn / LightGBM, Streamlit
